🤔 什麼是 awesome-codex-skills？

簡單說：它是一個 Codex CLI 的技能擴充包目錄。

OpenAI 的 Codex CLI 本身是一個在終端機運行的 AI coding agent，可以讀寫檔案、執行指令、自動完成開發任務。但原生的 Codex CLI 只能操作本機環境，無法直接呼叫外部服務。

awesome-codex-skills + Composio 解決了這個問題：透過預建的 skill 模組，你的 Codex Agent 可以：

  • 自動在 GitHub 開 Issue、建 PR、查 commit
  • 讀寫 Notion 資料庫、建立頁面
  • 發送 Slack 訊息、查詢頻道記錄
  • 操作 Gmail 收發信件
  • 串接 Jira、Linear、Trello 等專案管理工具
  • 呼叫任意 REST API（透過 Composio 的 HTTP 工具）
💡 一句話定位：awesome-codex-skills 是讓 Codex CLI 從「只會寫程式」升級成「能操作整個開發工作流」的橋樑。

🏗️ 架構概覽：Codex CLI + Composio 如何協作

整個系統由三層組成：

  1. Codex CLI（Agent 核心）：接收你的自然語言指令，規劃並執行任務
  2. awesome-codex-skills（技能目錄）：定義 Agent 可以呼叫哪些工具，以及如何呼叫
  3. Composio 平台（工具執行層）：實際處理 OAuth 認證、API 呼叫、錯誤重試

你只需要在 Composio 上完成一次 OAuth 授權（例如連接你的 GitHub 帳號），之後 Codex Agent 就能直接說「幫我在 repo X 開一個 bug issue」，系統自動完成所有 API 呼叫。

⚙️ 安裝步驟：5 分鐘完成環境設定

步驟 1：安裝 Codex CLI

npm install -g @openai/codex

步驟 2：安裝 Composio SDK

pip install composio-core composio-openai

步驟 3：Clone awesome-codex-skills

git clone https://github.com/ComposioHQ/awesome-codex-skills.git
cd awesome-codex-skills

步驟 4：設定 API 金鑰

export OPENAI_API_KEY="sk-..."
export COMPOSIO_API_KEY="your-composio-key"

Composio API Key 可在 app.composio.dev 免費申請。

步驟 5：連接你的工具（以 GitHub 為例）

composio add github

執行後會開啟瀏覽器進行 OAuth 授權，完成後 Composio 會儲存你的 token，之後 Agent 可直接使用。

✅ 完成！現在你的 Codex Agent 已經可以操作 GitHub。用同樣方式執行 composio add slack、composio add notion 來連接更多工具。

💡 進階技巧：自訂 Skill 擴充

如果 Composio 的 1,000+ 工具還不夠，你可以自己寫 skill：

# 自訂 skill 範例：串接台灣股市 API
from composio import ComposioToolSet, Action

toolset = ComposioToolSet()

# 定義自訂工具
@toolset.tool(name="tw_stock_price")
def get_tw_stock(ticker: str) -> dict:
    """查詢台灣股票即時價格"""
    import requests
    resp = requests.get(f"https://mis.twse.com.tw/stock/api/getStockInfo.jsp?ex_ch=tse_{ticker}.tw")
    return resp.json()

# 在 Codex Agent 中使用
tools = toolset.get_tools(actions=[Action.GITHUB_CREATE_ISSUE, "tw_stock_price"])

❓ 常見問題 FAQ

Q：awesome-codex-skills 和 mattpocock/skills 有什麼不同？

mattpocock/skills 是針對 Claude Code CLI 的技能包，專注在 coding workflow（TDD、PR 審查等）。awesome-codex-skills 則是針對 OpenAI Codex CLI，強調外部服務整合（GitHub、Slack、Notion 等）。兩者定位不同，可以同時使用。

Q：Composio 免費方案夠用嗎？

免費方案支援最多 3 個工具連接、每月 1,000 次 API 呼叫，個人開發者和小型專案完全夠用。如果是團隊使用或高頻自動化，再考慮升級付費方案。

Q：需要 OpenAI API Key 嗎？費用怎麼算？

是的，Codex CLI 需要 OpenAI API Key。費用依使用量計算，一般開發任務每次約 $0.01–$0.05，日常使用每月大約 $5–$20。建議設定 usage limit 避免超支。

Q：可以在 Windows 上使用嗎？

可以，但建議使用 WSL2（Windows Subsystem for Linux）以獲得最佳相容性。原生 Windows PowerShell 環境可能遇到路徑和權限問題。

Q：安全性如何？OAuth token 存在哪裡？

Composio 使用業界標準的 OAuth 2.0，token 加密儲存在 Composio 的雲端服務。如果你有資安顧慮，可以使用 Composio 的 self-hosted 版本，把所有 token 存在自己的伺服器上。
