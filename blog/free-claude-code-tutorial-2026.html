🤖什麼是 free-claude-code？

free-claude-code 是 GitHub 上一個開源專案（作者：Alishahryar1），讓你透過 NVIDIA NIM API 作為代理後端，完全免費使用 Anthropic Claude Code 的全部功能——包含程式碼生成、重構、除錯、測試撰寫，以及 Agent 模式的自主任務執行。

原版 Claude Code 需要 Anthropic API 金鑰，按 token 計費，重度使用者每月可能花費 $50–$200 美元。free-claude-code 的核心思路是：把 API 請求路由到 NVIDIA NIM 的免費額度，讓你在不花一毛錢的情況下，享受同等的 AI 編程體驗。

✅ 核心優勢一句話

NVIDIA NIM 提供免費 API 額度（每月數百萬 tokens），free-claude-code 把這個額度接進 Claude Code 的介面，讓你零成本使用業界最強的 AI 編程工具。

🚀5 步完整安裝教學

以下教學適用於 macOS、Linux 及 Windows（WSL2）。預計完成時間：5 分鐘。

1

確認環境需求

需要 Node.js 18+ 和 npm。執行以下指令確認版本：

node --version   # 需要 v18 以上
npm --version    # 需要 v8 以上

若尚未安裝 Node.js，前往 nodejs.org 下載 LTS 版本。

2

申請 NVIDIA NIM API 金鑰（免費）

前往 build.nvidia.com，登入後建立 API Key，複製備用。

💡 NVIDIA NIM 免費額度說明

新帳號每月提供數百萬 tokens 免費額度，足夠個人開發日常使用。超出後可選擇付費升級，但大多數用戶不會超出免費上限。

3

安裝 Claude Code CLI

若尚未安裝官方 Claude Code，先執行：

npm install -g @anthropic-ai/claude-code
4

安裝 free-claude-code 代理

從 GitHub 克隆並安裝：

git clone https://github.com/Alishahryar1/free-claude-code.git
cd free-claude-code
npm install
5

設定環境變數並啟動

建立 .env 檔案，填入 NVIDIA NIM API 金鑰，然後啟動代理伺服器：

# 建立 .env 檔案
echo "NVIDIA_NIM_API_KEY=your_api_key_here" > .env

# 啟動代理伺服器（預設 port 3000）
npm start

看到 Proxy server running on port 3000 表示啟動成功。接著在另一個 Terminal 視窗，設定 Claude Code 使用本地代理：

export ANTHROPIC_BASE_URL=http://localhost:3000
export ANTHROPIC_API_KEY=dummy-key
claude

💻VSCode 整合設定

free-claude-code 完整支援 VSCode 整合，讓你在編輯器內直接呼叫 AI 協助。

方法一：透過 Terminal 面板使用

在 VSCode 內開啟 Terminal（Ctrl+`），設定環境變數後直接執行 claude：

export ANTHROPIC_BASE_URL=http://localhost:3000
export ANTHROPIC_API_KEY=dummy-key
claude "幫我重構這個函數，加上 TypeScript 型別"

方法二：設定 VSCode 工作區環境變數

在專案根目錄建立 .vscode/settings.json，讓每次開啟專案自動套用設定：

{
  "terminal.integrated.env.linux": {
    "ANTHROPIC_BASE_URL": "http://localhost:3000",
    "ANTHROPIC_API_KEY": "dummy-key"
  },
  "terminal.integrated.env.osx": {
    "ANTHROPIC_BASE_URL": "http://localhost:3000",
    "ANTHROPIC_API_KEY": "dummy-key"
  }
}

方法三：加入 shell 設定檔（推薦）

把環境變數加入 ~/.zshrc 或 ~/.bashrc，讓代理伺服器啟動後全域生效：

# 加入 ~/.zshrc
echo 'export ANTHROPIC_BASE_URL=http://localhost:3000' >> ~/.zshrc
echo 'export ANTHROPIC_API_KEY=dummy-key' >> ~/.zshrc
source ~/.zshrc

💡 讓代理伺服器背景常駐

使用 pm2 讓代理伺服器在背景持續運行，不需要每次手動啟動：

npm install -g pm2
cd free-claude-code
pm2 start npm --name "free-claude-proxy" -- start
pm2 save && pm2 startup

❓常見問題 FAQ

free-claude-code 合法嗎？會違反 Anthropic 使用條款嗎？
free-claude-code 本身是開源工具，使用 NVIDIA NIM 的合法 API 服務。它不是破解或繞過付費牆，而是使用另一個合法的 API 提供商。NVIDIA NIM 有自己的服務條款，請確保你的使用符合 NIM 的規定。
NVIDIA NIM 免費額度夠用嗎？
對於個人開發者和學習用途完全夠用。NVIDIA NIM 新帳號提供數百萬 tokens 的免費額度，一般的程式碼生成、重構、除錯任務每次消耗約 1,000–5,000 tokens，每月免費額度可支撐數百次操作。
Windows 用戶可以使用嗎？
可以，但建議透過 WSL2（Windows Subsystem for Linux）使用，體驗最佳。直接在 Windows PowerShell 使用也可行，但部分 shell 指令可能需要調整。
遇到 "Connection refused" 錯誤怎麼辦？
確認代理伺服器正在運行（npm start 在 free-claude-code 目錄），且 ANTHROPIC_BASE_URL 設定為 http://localhost:3000。如果 port 3000 被佔用，可在 .env 中修改 PORT=3001 並更新環境變數。
free-claude-code 和官方 Claude Code 的回應品質有差異嗎？
有輕微差異。NVIDIA NIM 使用的模型版本可能略舊於 Anthropic 最新版，但對於日常開發任務（程式碼生成、重構、除錯）品質差異不明顯。如果你需要最新模型能力，建議使用官方付費版本。
可以在 CI/CD 流程中使用嗎？
技術上可行，但不建議在生產 CI/CD 中使用，原因是免費額度的速率限制可能導致流程中斷。CI/CD 環境建議使用官方 API 並設定適當的 token 預算。
