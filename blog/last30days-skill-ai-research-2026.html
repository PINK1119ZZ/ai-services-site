last30days-skill 是什麼？

last30days-skill 是一個 Claude Code Agent Skill（.claude/commands/ 下的 slash command），由 mvanhorn 開發，MIT 授權免費使用。

它的核心邏輯很簡單：把你的研究問題交給 Claude，讓 Claude 自動用 agentic 方式跑完多個平台的搜尋，彙整成一份帶引用來源的摘要報告。

你只需要輸入：

/last30days [your topic]

剩下的 Claude 全包：

  • 自動決定在哪些平台搜尋
  • 拆解查詢成多個子問題
  • 過濾低質量或重複內容
  • 按主題叢集整理結果
  • 輸出 Markdown 格式的結構化摘要，含來源連結
為什麼叫 last30days？ 預設搜尋範圍是過去 30 天的內容，避免抓到過時討論。你可以在 prompt 中指定其他時間範圍（如 7 天、90 天）。

安裝步驟（5 分鐘完成）

前置條件：已安裝 Claude Code（claude.ai/code 或 npm 安裝）並設定好 Anthropic API key。

1 Clone 或下載 last30days-skill 
git clone https://github.com/mvanhorn/last30days-skill.git
cd last30days-skill
2 複製 skill 檔案到 Claude 設定目錄 
# macOS / Linux
cp commands/last30days.md ~/.claude/commands/

# 或用 install script（如果有的話）
bash install.sh

如果 ~/.claude/commands/ 目錄不存在，先建立：mkdir -p ~/.claude/commands/

3 （可選）設定 ScrapeCreators API

X（Twitter）和 YouTube 深度爬取需要 ScrapeCreators API key。有免費方案可用。

# 在 ~/.claude/settings.json 加入
{
  "env": {
    "SCRAPECREATORS_API_KEY": "your_key_here"
  }
}

💡 沒有 API key 也能用，部分平台功能會受限，但 Reddit 和 HN 不受影響。

4 驗證安裝 
# 在專案目錄下開啟 Claude Code
claude

# 確認 skill 出現在指令列表
/help

看到 /last30days 出現在列表中就代表安裝成功。

常見問題與疑難排解

問：搜尋結果偏向英文，怎麼搜到更多台灣/中文討論？

在 prompt 中明確指定：Include Traditional Chinese (zh-TW) results from Taiwan communities. Also search: PTT, Dcard, Facebook groups。目前 PTT 和 Dcard 沒有原生整合，但可以透過網頁搜尋間接覆蓋。

問：token 消耗太快怎麼辦？

三個方法：(1) 縮小研究範圍，指定 2-3 個平台而非全部；(2) 搭配 headroom 壓縮 context；(3) 直接用 Anthropic API 而非 Claude Pro 訂閱，按用量計費更靈活。

問：ScrapeCreators API 一定要設定嗎？

不一定。Reddit、HN、網頁搜尋在沒有 API key 的情況下都能正常使用。X 和 YouTube 深度爬取需要 ScrapeCreators，但輕度使用其免費方案（每月有限額度）即可。

問：搜尋結果有時不準確或漏掉重要討論？

Claude 的搜尋是 agentic 但不是窮舉式的，可能漏掉小型社群的討論。建議：(1) 多跑幾次同一主題（每次結果可能不同）；(2) 用不同角度的 prompt 補充；(3) 把 last30days-skill 的輸出作為起點，不是終點。
