你是否遇過這種情況：LINE 官方帳號一天收到幾十則詢問，問題幾乎都是一樣的——「你們幾點開門？」「有素食嗎？」「可以預約嗎？」每次都要手動回覆，累死人也讓客人等很久。其實 LINE 官方帳號內建的「自動回覆」功能，可以幫你解決大部分重複問題，完全不需要工程師，自己就能設定好。

LINE 官方帳號自動回覆功能介紹

LINE 官方帳號（LINE Official Account，簡稱 LINE OA）是 LINE 官方提供給商家使用的商業帳號，你可以在上面發送廣播訊息、管理好友名單，以及設定自動回覆。

自動回覆功能主要分為三種：

  • 關鍵字回應：客人輸入特定關鍵字，帳號自動回覆對應訊息
  • 歡迎訊息：客人加入好友的第一則訊息
  • 非營業時間訊息：超出設定的營業時間，自動告知客人

重要提醒：LINE 官方帳號的自動回覆屬於「固定式」回覆，只能設定關鍵字對應固定訊息。如果你需要 AI 對話、預約系統或串接資料庫，那就需要升級到 LINE Bot 開發服務。

LINE 官方帳號自動回覆設定步驟

以下教學適用於 LINE Official Account Manager（電腦版後台）。手機版 LINE OA App 操作路徑略有不同，但功能相同。

1

登入 LINE Official Account Manager

前往 manager.line.biz，使用你的 LINE 帳號登入，選擇你的官方帳號。

2

進入「自動回應訊息」設定

左側選單點選「訊息」→「自動回應訊息」。如果看不到此選項，請確認你有管理員或編輯者權限。

3

選擇回應模式

系統提供兩種模式：智慧聊天（LINE AI 自動判斷，答案較通用）和手動設定關鍵字回應。建議選擇「手動設定」，回覆內容更精準符合你的業務。

4

新增關鍵字回應

點選「建立」，輸入關鍵字（如「營業時間」、「價格」）和對應的回覆訊息。一組關鍵字最多可包含 5 個相似詞，例如「幾點開」、「幾點營業」、「開門時間」都可以對應同一則回覆。

5

啟用並測試

設定完成後，記得將狀態切換為「啟用」。然後用自己的個人 LINE 帳號加入官方帳號測試，確認回覆是否正常觸發。

三種自動回覆類型詳解

① 關鍵字回應：最常用的自動回覆

關鍵字回應是最核心的自動回覆功能。設定邏輯很簡單：「當客人說 X，自動回覆 Y」。以美容院為例，可以設定以下常見問題：

關鍵字組 包含詞彙範例 自動回覆內容
營業時間 幾點開、幾點到幾點、營業 「我們週一至週六 10:00-20:00，週日公休 🗓」
價格/收費 多少錢、費用、價格、收費 「歡迎點選選單查看完整服務價目 👇」+ 圖文訊息
預約 預約、約時間、訂位、能預約嗎 「請點選下方預約連結，或直接傳訊息給我們 📅」
地址/位置 在哪、地址、怎麼去、捷運 「地址：台北市〇〇路〇〇號，靠近〇〇捷運站」+ 地圖連結
停車 停車、停車場、車位 「店家附近有〇〇停車場，步行約 3 分鐘 🚗」

設定小技巧：每組關鍵字最多設 5 個相似詞，盡量涵蓋客人的各種說法。關鍵字不需要完全符合，只要訊息中包含關鍵字就會觸發回覆。

② 歡迎訊息：第一印象決定留存率

歡迎訊息是客人加入好友後，自動發送的第一則訊息。這是你留住新好友的黃金機會，一定要好好設計。

好的歡迎訊息應該包含：

  • 品牌問候語（簡短、有溫度）
  • 你能提供什麼服務（30 字以內說清楚）
  • 引導下一步行動（查看選單、預約、優惠）
  • 可搭配一張精美的圖片或圖文選單（Rich Menu）

歡迎訊息範例（美容沙龍）：

「嗨！感謝你加入〇〇美容沙龍 ✨

我們提供剪髮、染髮、護髮等全系列服務，讓你每次出門都美美的！

👇 點選下方選單查看服務價格與預約方式
有任何問題也歡迎直接傳訊給我們～」

③ 非營業時間訊息：讓客人不白等

設定營業時間後，當客人在下班後或假日傳訊息，系統會自動回覆告知目前無法即時回覆，同時可以引導客人自助取得資訊。

非營業時間訊息設定位置：「聊天」→「回應設定」→「非回應時間」。

非營業時間訊息建議內容：

  • 說明目前非營業時間
  • 告知何時會回覆
  • 提供自助資訊（如：常見問題、預約連結）

注意：非營業時間訊息只有在「回應模式」設定為「聊天」時才會生效。如果是「Bot」模式，LINE 不提供非營業時間訊息功能，這種情況建議透過 LINE Bot 開發 來處理。

什麼時候該升級到 LINE Bot？

LINE 官方帳號的內建自動回覆功能對小店家來說已經非常夠用，但以下幾種情況，你可能需要考慮升級到正式的 LINE Bot：

情況 1：客人問的問題五花八門，固定關鍵字不夠用

如果你每天收到的客人問題非常多樣，無法用有限的關鍵字涵蓋，那 AI 智慧對話的 LINE Bot 就能派上用場。它可以理解自然語言，就算客人問法不一樣，都能給出合理的回覆。

情況 2：需要在 LINE 內完成預約或訂單

如果你的業務依賴預約（美容院、診所、補習班、餐廳），而且希望客人直接在 LINE 裡完成預約，而不是被導到外部網站，LINE Bot 搭配 LIFF 就是最佳解法。詳細可參考 LINE 預約系統完整介紹。

情況 3：需要自動記錄客戶資料

如果你希望每次客人查詢或預約時，資料自動存到 Google Sheets 或你的 CRM，LINE Bot 可以幫你完全自動化這個流程，省去手動記錄的麻煩。

情況 4：每月訊息量超過 1,000 則

免費版 LINE 官方帳號每月只有 500 則推播，當你的好友數達到一定規模後，每月訊息費用會大幅上升。搭配 LINE Bot 的彈性訊息管理，可以更有效地使用訊息配額。

情況 5：想要 24 小時不間斷的真正智慧回覆

LINE OA 的自動回覆是固定的，碰到不在關鍵字清單裡的問題就沉默。LINE Bot 搭配 AI 客服可以讓你的帳號真正做到 24 小時都能給出有幫助的回應，大幅提升客戶滿意度。想了解更多可參考 AI 客服完整指南。

評估建議：如果你的 LINE 官方帳號每週需要回覆超過 50 則重複問題，或者客人常常因為沒有即時回覆而流失，投資一套 LINE Bot 通常 3-6 個月就能回本。歡迎 免費諮詢，我們幫你評估是否值得升級。

LINE 官方帳號自動回覆常見錯誤與解法

錯誤 1：關鍵字設太長

很多人設定關鍵字時會寫「你們幾點開門」這種完整句子，但實際上客人可能說「幾點開」或「幾點開始」。關鍵字應該設成短詞，讓系統更容易匹配，例如只設「幾點」、「開門」、「營業」。

錯誤 2：沒有設定歡迎訊息

很多小店家忽略了歡迎訊息的重要性。新好友加入後，如果沒有任何訊息，會讓人感覺這個帳號「沒有在管」，容易直接封鎖。一定要設定一則溫馨的歡迎訊息。

錯誤 3：回覆訊息太長

LINE 的自動回覆訊息如果太長，客人不會看完。重要資訊要在前 3 行內呈現，搭配 emoji 增加可讀性。長內容可以改用圖文訊息（Message Template）呈現。

錯誤 4：沒有定期更新關鍵字

店家的資訊會隨時更改——價格調整、地址搬遷、服務項目更新。記得定期（至少每季）檢查自動回覆內容是否還正確。

進一步優化 LINE 官方帳號的 3 個技巧

技巧 1：搭配 Rich Menu 圖文選單

Rich Menu（圖文選單）是 LINE 官方帳號的底部選單，可以設定快速連結按鈕。結合自動回覆，讓客人不需要輸入關鍵字，直接點選按鈕就能取得資訊。Rich Menu 可以在後台免費設定，或請設計師製作更精美的版本。

技巧 2：善用圖文訊息

自動回覆不只能發文字，也可以發圖文訊息（包含圖片和連結按鈕）。例如在「價格」關鍵字的回覆中，附上一張精美的服務菜單圖片，效果遠比純文字好。

技巧 3：追蹤哪些關鍵字最常被觸發

LINE 官方帳號後台有訊息統計功能，可以看到哪些自動回覆最常被觸發。高觸發率的關鍵字代表客人最常問的問題，你可以把這些資訊用來優化服務或做行銷素材。

常見問題

LINE 官方帳號自動回覆要怎麼設定？
登入 LINE Official Account Manager（manager.line.biz），進入「訊息」→「自動回應訊息」，選擇關鍵字回應模式，即可新增關鍵字和對應的自動回覆訊息。免費帳號也可使用此功能。
LINE 官方帳號自動回覆免費嗎？
LINE 官方帳號的基本自動回覆（關鍵字回應、歡迎訊息）是免費功能。但每月主動推播訊息有數量限制，免費版每月 500 則，超出需升級付費方案（輕用量方案 NT$800/月起）。
LINE 官方帳號自動回覆和 LINE Bot 有什麼差別？
LINE 官方帳號內建的自動回覆屬於「固定式」，只能設定關鍵字對應固定回覆，最多 200 組。LINE Bot 是透過 Messaging API 開發的進階機器人，可以做 AI 對話、串接資料庫、預約系統等複雜功能，沒有關鍵字數量限制。
LINE 官方帳號可以設定幾組關鍵字自動回覆？
目前 LINE 官方帳號管理後台最多可設定 200 組關鍵字回應，每組關鍵字可包含最多 5 個同義詞，對應一則回覆訊息（文字、圖片或圖文訊息）。
什麼情況下需要從 LINE 官方帳號升級到 LINE Bot？
當你需要 AI 智慧對話、自動預約排班、串接 Google Sheets 或 CRM、或是需要更複雜的流程邏輯時，就需要升級到 LINE Bot。通常每週回覆超過 50 則重複問題，或客戶問題多樣性高的店家，升級效益最大。
