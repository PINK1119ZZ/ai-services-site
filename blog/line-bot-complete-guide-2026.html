為什麼台灣中小企業需要 LINE Bot？

LINE 在台灣的覆蓋率超過 95%，幾乎每個成年用戶都有 LINE 帳號。對中小企業而言，這意味著：

  • 客戶溝通成本最低：用戶不需要下載新 App、不用註冊新帳號，加好友就能互動
  • 24 小時自動回覆：不漏單、不錯失即時詢問，比人工客服便宜 10 倍
  • 跟現有業務系統整合：可以串接 Google Sheets、CRM、訂單系統、AI 模型
  • 群發成本可控：LINE 官方帳號計費透明，比簡訊群發便宜 5-10 倍

實際應用情境包括：餐廳預約、診所掛號、美髮預約、電商客服、訂單通知、會員管理、課程報名等。商用 AI 聊天機器人怎麼選有更詳細的場景對應分析。

LINE Bot 的 4 大類型

很多人把 LINE 上的所有自動化都叫「LINE Bot」，但實際上有 4 種完全不同的技術選型，費用跟能做到的事差很大：

類型 1：LINE 官方帳號內建自動回覆

LINE 官方帳號（LINE Official Account, OA）內建的「自動回應訊息」「關鍵字回應」「圖文選單」功能。設定 5 分鐘就能用，不需要工程師。

適合場景：基本問答（營業時間、地址、菜單）、群發優惠券、簡單的關鍵字回覆。

限制：無法 AI 對話、無法串接你的資料庫、無法做預約 / 訂單流程。

費用：LINE OA 月費（依通話量）NT$0-1,000/月。沒有開發費。

類型 2：Messaging API（自製 LINE Bot）

用 LINE 提供的 Messaging API 開發自己的 Bot 邏輯。可以接收用戶訊息、回覆訊息、推送通知、處理圖片影片、生成 Flex Message 卡片。

適合場景：客服 FAQ Bot、訂單通知、AI 對話、簡單的指令式互動。

限制：複雜的表單流程在純訊息互動裡很難做（要靠 LIFF）。

費用：基本 Bot 開發 NT$8,000 起，含雲託管月費 NT$1,500 起。

類型 3：LIFF（LINE Front-end Framework）

LIFF 讓你在 LINE 內部開啟一個自製網頁，能用完整的 HTML/JS 寫複雜表單、購物車、會員系統，但保有 LINE 用戶識別。

適合場景：預約表單（時間選擇、人數、備註）、會員資料填寫、購物車結帳、複雜流程。

限制：要懂前端開發。

費用：LIFF 表單 NT$5,000-15,000，含後端整合 NT$15,000-30,000。

類型 4：AI 整合（GPT / Claude / Gemini）

把 LINE Bot 接上 AI 模型，做真正的對話式客服、產品推薦、智慧 FAQ。可選 OpenAI GPT、Anthropic Claude、Google Gemini，或自架 LLM。

適合場景：智慧客服、自然語言訂單查詢、AI 商品顧問、多語言支援。

限制：API 費用 + 需要設計 prompt 跟知識庫。

費用：AI 客服基本款 NT$15,000-25,000，企業級 RAG 知識庫整合 NT$30,000+，每月 API 費用 NT$500-3,000。詳見 LINE AI 智慧客服實作。

4 類型怎麼選

  • 只需要基本回覆 → 類型 1（LINE OA 內建）
  • 要做客製功能但不複雜 → 類型 2（Messaging API）
  • 要做表單 / 預約 / 購物 → 類型 3（LIFF）
  • 要 AI 對話 / 自動回答 → 類型 4（AI 整合）

常見組合

  • 「預約 + 自動提醒」= 類型 2 + 3
  • 「AI 客服 + 訂單查詢」= 類型 2 + 4
  • 「電商 + AI 顧問 + LIFF 結帳」= 類型 2 + 3 + 4

該找代客開發還是自己做？

適合自己做的情況

  • 已有工程師團隊或會 Python / Node.js
  • 需求簡單（只要自動回覆、不需要整合）
  • 願意花 2-4 週學習成本
  • 之後要持續迭代 / 改功能

適合找代客的情況

  • 沒有工程師或工程師不熟 LINE
  • 需要快速上線（1-2 週內）
  • 需要整合 AI / Google Sheets / CRM / 金流
  • 需要長期雲託管 + 維護

多數中小企業老闆面臨的真正問題不是「能不能自己做」，而是「自己做的時間成本能不能接受」。如果你的時間值 NT$1,000/小時，自學 + 開發 100 小時 = NT$100,000 機會成本，還不如花 NT$15,000 找代客 2 週做完。

5 個常見錯誤

錯誤 1：直接買套裝 SaaS，事後才發現綁死

很多 SaaS LINE Bot 平台月費低、上手快，但客製化空間有限，數據在他們手上，要轉移很痛苦。建議：簡單需求用 SaaS OK，但確保預約紀錄 / 客戶資料能匯出。

錯誤 2：跳過 LIFF，用純訊息做複雜表單

讓用戶在 LINE 訊息裡一問一答填 10 個欄位 = 體驗災難。複雜表單一律用 LIFF，5 分鐘設計 vs 2 週調 UX。

錯誤 3：上線後沒監控，掛掉幾天才發現

LINE Bot 是用戶第一線，掛了你不知道，客戶以為你不回。一定要設「Bot 健康監控 + Telegram 告警」（我們的 雲託管方案 內建這個）。

錯誤 4：AI 串接沒設成本上限

OpenAI / Claude API 計費按 token，沒設月費上限可能爆走。實作時一定要：用戶頻次限制 + 月度總成本上限 + 異常 token 告警。詳見 AI Token 成本計算器。

錯誤 5：只做 LINE，忽略多平台複用

同樣的客服 Bot 邏輯如果只在 LINE 跑，等你想擴展到 Telegram / Web Chat 又要重做。建議從一開始就用平台無關的核心 + 各平台 adapter。

常見問題

Q1：LINE Bot 跟 LINE 官方帳號差在哪？

LINE 官方帳號是 LINE 提供的標準帳號功能（群發、自動回覆、優惠券等），月費基於通話量。LINE Bot 是用 Messaging API 自製功能（預約、AI 對話、訂單查詢），可以串接你的後端系統，月費低但有開發成本。

Q2：LINE Bot 開發費用大約多少？

基本 LINE Bot（自動回覆 + 簡單功能）NT$8,000 起，含 AI 客服或預約系統 NT$15,000-25,000，企業級多模組整合 NT$30,000+。每月雲託管費用 NT$1,500 起。

Q3：LINE Bot 開發要多久？

從需求確認到上線最快 2 週。基本功能 1-2 週，含複雜整合（CRM / Google Sheets / 金流）3-4 週。

Q4：LIFF 是什麼？什麼時候要用？

LIFF（LINE Front-end Framework）讓你在 LINE 內部開啟自製網頁。需要表單填寫、會員資料、複雜流程時用 LIFF（例如預約表單、購物車）。

Q5：沒有工程師可以自己做 LINE Bot 嗎？

簡單自動回覆可以用 LINE 官方帳號內建設定，不需要工程師。但若要 AI 對話、預約系統、串接你的後端 — 找代客開發比較快也比較穩，省下學習成本。

Q6：每月要多少錢維護？

雲主機 NT$1,500-5,000/月（看流量）+ AI API NT$500-3,000/月（如果有用 AI）+ 維護費（自己做 0、找代客 NT$2,000-8,000/月）。

Q7：可以跟我現有的 Google Sheets / Excel / CRM 整合嗎？

可以。Google Sheets 整合很常見（用 Apps Script 或 API），CRM 看你用哪家（Salesforce、HubSpot、Pipedrive 都有 API），Excel 通常匯出成 Sheets 再串。
