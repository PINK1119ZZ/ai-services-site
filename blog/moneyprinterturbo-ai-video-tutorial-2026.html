安裝教學：Python 版本（推薦新手）

系統需求

  • Python 3.10 或以上
  • ffmpeg（影音處理必要工具）
  • 至少 4GB RAM（8GB 以上更佳）
  • OpenAI API key 或其他 AI 服務 key（腳本生成需要）

Step 1：安裝 ffmpeg

macOS：

brew install ffmpeg

Ubuntu / Debian：

sudo apt update && sudo apt install ffmpeg -y

Windows： 至 ffmpeg.org 下載，加入 PATH 環境變數。

Step 2：Clone 並安裝

git clone https://github.com/harry0703/MoneyPrinterTurbo.git cd MoneyPrinterTurbo # 建立虛擬環境（強烈建議） python3 -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate # 安裝依賴 pip install -r requirements.txt

Step 3：設定 config.toml

複製範例設定檔並填入你的 API key：

cp config.example.toml config.toml

開啟 config.toml，主要需要設定的部分：

[app] # LLM 設定（腳本生成） llm_provider = "openai" # 或 "azure", "gemini", "ollama" openai_api_key = "sk-..." openai_model_name = "gpt-4o" # TTS 設定（配音） voice_provider = "azure" # 或 "openai", "google", "pyttsx3"（免費） azure_tts_key = "..." azure_tts_region = "eastasia" # 香港/東亞節點速度較快 # 素材搜尋 pexels_api_key = "..." # 免費申請：pexels.com/api
💡 零成本快速試用： 把 llm_provider 設為 ollama（本地模型免費），voice_provider 設為 pyttsx3（本地 TTS 免費），不需要任何 API key 就能跑起來。音質和腳本品質會低一些，但適合先熟悉工作流程。

Step 4：啟動 Web UI

python webui.py

開啟瀏覽器訪問 http://localhost:8501，就能看到 MoneyPrinterTurbo 的 Web 操作介面。

安裝教學：Docker 版本（推薦進階使用者）

如果你不想污染本機 Python 環境，Docker 是更乾淨的選擇：

git clone https://github.com/harry0703/MoneyPrinterTurbo.git cd MoneyPrinterTurbo # 複製設定檔 cp config.example.toml config.toml # （同上，填入你的 API key） # 啟動容器 docker-compose up -d # 查看日誌 docker-compose logs -f

啟動後同樣訪問 http://localhost:8501。

⚠️ 注意： Docker 版本的首次啟動需要下載 Whisper 模型（約 300MB-1GB），需要有耐心等待。建議在網路穩定的環境下操作。

5 分鐘快速上手：產出你的第一支影片

環境設定好後，用 Web UI 生成第一支影片只需要幾個步驟：

  1. 選擇語言： 選「繁體中文」
  2. 輸入主題： 例如「台灣夜市必吃小吃 TOP 5」或「2026 年最值得學的 AI 技能」
  3. 設定影片長度： 短影音建議 30-60 秒，長影音可設 2-5 分鐘
  4. 選擇影片比例： 9:16（TikTok/Shorts）或 16:9（YouTube）
  5. 選擇聲音： 從 Azure/Google TTS 選一個繁中女聲或男聲
  6. 點擊「生成影片」 等待 AI 完成所有步驟

整個過程大約 5-10 分鐘（主要耗時在 Whisper 字幕生成），完成後可以在介面上預覽並下載 MP4 檔案。

常見問題與故障排除

ffmpeg 找不到怎麼辦？

# 確認 ffmpeg 是否正確安裝 ffmpeg -version # 如果提示找不到，確認 PATH 設定 which ffmpeg # Unix where ffmpeg # Windows

Whisper 模型下載很慢？

預設會下載 base 模型，已經有不錯的準確度。如果要更高準確度可以換 medium 或 large，但速度更慢。在 config.toml 設定：

whisper_model_name = "base" # 快速，準確度普通 # whisper_model_name = "medium" # 較慢，繁中準確度更高

Azure TTS 找不到繁中語音？

Azure TTS 的台灣繁中語音 voice name 是 zh-TW-HsiaoChenNeural（女聲）或 zh-TW-YunJheNeural（男聲），在 config.toml 指定：

voice_name = "zh-TW-HsiaoChenNeural"

生成的影片沒有聲音？

通常是 ffmpeg 版本問題或 TTS 設定有誤。確認：(1) ffmpeg 版本 >= 4.0；(2) TTS API key 有效且配額足夠；(3) 語音檔案有成功生成（查看 temp/ 目錄下是否有 .mp3 檔）。

FAQ

Q1：MoneyPrinterTurbo 是免費的嗎？

MoneyPrinterTurbo 本身是開源免費（MIT 授權），但需要 OpenAI / Azure TTS / 其他 AI 服務的 API key，這些服務通常按用量計費。如果你用免費的 pyttsx3 TTS 和本地 Ollama 模型可以完全零成本，但音質和腳本品質較差。建議新手從免費選項開始試，確認工作流程後再升級付費服務。

Q2：MoneyPrinterTurbo 支援繁中字幕嗎？

支援！MoneyPrinterTurbo 使用 OpenAI Whisper 自動生成字幕，Whisper 對繁體中文的辨識率相當高。字幕生成後可以手動調整，也可以設定字幕樣式（字體、顏色、位置）。

Q3：沒有高階 GPU 也能用嗎？

可以！MoneyPrinterTurbo 可以跑在 CPU 模式，不需要 GPU。但本地 Whisper 轉錄速度會慢一些。最快的方式是用 OpenAI Whisper API（付費但快速），或把整個工具部署在 DigitalOcean GPU Droplet 上。

Q4：生成的影片可以直接上傳 YouTube 嗎？

技術上可以，但不建議未經確認就上傳。AI 自動生成的影片需要人工檢查：確認字幕準確性、音頻與畫面同步、背景影片是否有版權疑慮（建議使用 Pexels/Pixabay 免費素材）。通常需要 5-10 分鐘的最終確認。

Q5：MoneyPrinterTurbo 和 CapCut AI 的差別是什麼？

MoneyPrinterTurbo 的優勢是完全自動化（輸入主題→成片），且開源免費可自訂。CapCut AI 提供更好的剪輯介面和更豐富的模板，適合需要精緻化後製的影片。最佳組合：用 MoneyPrinterTurbo 快速產出素材，再用 CapCut 做最終精修。
