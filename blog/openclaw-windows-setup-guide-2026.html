🪟 OpenClaw Windows 是什麼？

OpenClaw 是一個開源的 AI 個人助理框架（Gateway），讓你可以把 Claude、GPT-5、Gemini 等大語言模型接進你的日常工作流——回訊息、寫程式、查資料、設定提醒，全部自動化。

openclaw-windows-node 是 OpenClaw 的 Windows 專屬 companion suite，包含三個元件：

🖥️

System Tray 常駐

OpenClaw Gateway 在背景運行，工具列右下角一個小圖示，隨時可呼叫，不佔主視窗空間。

📱

Node 手機配對

掃 QR Code 把手機（iOS/Android）配對成 OpenClaw Node，讓 AI 能讀取手機通知、相機、位置。

⌨️

PowerToys 整合

整合 Windows PowerToys Command Palette，按 Win+空白鍵立刻呼叫 AI，不用切換任何視窗。

簡單說：它讓 OpenClaw 在 Windows 上的使用體驗，變得跟 Mac 一樣流暢，甚至因為 PowerToys 整合，某些情境下比 Mac 更好用。

📋 安裝前準備

在開始之前，確認你的環境符合以下條件：

需求 最低版本 建議
作業系統 Windows 10 64-bit Windows 11 23H2+
Node.js v18 v22 LTS
PowerToys（選用） v0.88 最新版（Microsoft Store）
AI 模型 任一 API Key Claude Sonnet 4.5 或 Ollama 本地模型
手機配對（選用） iOS 16 / Android 12 最新 iOS 或 Android
💡 沒有 API Key？ 可以先用 Ollama 在本機跑免費模型（例如 Llama 3.2 或 Qwen 2.5），完全不需要任何 API 費用，安裝完再換成 Claude。

🚀 安裝步驟：從零到跑起來

步驟一：安裝 Node.js

前往 nodejs.org，下載 v22 LTS（Windows Installer .msi），一路 Next 安裝完成。

安裝後開啟 PowerShell，驗證：

node --version
# 應顯示 v22.x.x
npm --version
# 應顯示 10.x.x

步驟二：安裝 OpenClaw CLI

npm install -g openclaw

這會安裝 OpenClaw Gateway 主程式。安裝完成後初始化：

openclaw init

按照提示輸入你的 AI 模型設定（可先跳過，之後再設定）。

步驟三：安裝 Windows Companion Suite

# 安裝 openclaw-windows-node
npm install -g openclaw-windows-node

# 啟動 System Tray 模式
openclaw-windows start --tray

執行後，工作列右下角（系統匣）應出現 OpenClaw 圖示。右鍵點擊可以：

  • 開啟 OpenClaw 聊天視窗
  • 查看目前連線的 Nodes（手機）
  • 重啟 Gateway
  • 查看 Logs

⚙️ 設定開機自動啟動

在 PowerShell（以系統管理員身分執行）：

openclaw-windows install-service

這會把 OpenClaw 註冊為 Windows 服務，開機自動啟動，不需要每次手動執行。

步驟四：設定 AI 模型

開啟 OpenClaw 設定（右鍵系統匣圖示 → Settings，或直接編輯 ~/.openclaw/config.json）：

{
  // 使用 Claude（建議）
  "model": "claude-sonnet-4-5",
  "apiKey": "sk-ant-xxxxxxxx",

  // 或使用本地 Ollama（免費）
  // "model": "ollama/llama3.2",
  // "baseUrl": "http://localhost:11434"
}
⚠️ API Key 安全提醒： config.json 存在本機，請確保不要把這個檔案 commit 進 Git repo 或分享給他人。OpenClaw 不會把你的 Key 傳送到任何外部伺服器。

📱 Node 配對：把手機接入 AI

OpenClaw 的 Node 系統讓你可以把手機（iOS/Android）配對成一個感測器節點——AI 就能看到你的通知、呼叫相機、甚至知道你在哪裡。

配對步驟

1. 在 Windows 上顯示配對 QR Code

右鍵系統匣圖示 → Pair New Node，螢幕出現 QR Code 和 6 位數配對碼。

2. 在手機上安裝 OpenClaw App

iOS：App Store 搜尋 "OpenClaw"。Android：Google Play 搜尋 "OpenClaw Node"。

3. 掃碼配對

打開 App → Scan QR Code，掃描 Windows 上的 QR Code。配對完成後，手機會出現在 OpenClaw 的 Nodes 列表中。

配對成功後，你可以在 OpenClaw 聊天視窗中問：

  • 「我的手機有什麼未讀通知？」
  • 「幫我拍一張照片分析一下這份文件」
  • 「我現在在哪裡？附近有什麼餐廳？」

⌨️ PowerToys 整合：Win+空白鍵呼叫 AI

這是 Microsoft Build 2026 Demo 中最讓現場觀眾驚豔的功能：不需要切換任何應用程式，直接在 Windows 的 Command Palette 裡呼叫 AI。

設定 PowerToys 整合

1. 安裝 PowerToys（如果還沒有）：

# 用 winget 安裝（最簡單）
winget install Microsoft.PowerToys

或從 Microsoft Store 搜尋 "PowerToys" 安裝。

2. 啟用 Command Palette Extension：

# 在 PowerShell 執行（以系統管理員身分）
openclaw-windows install-powertoys-extension

這個指令會自動把 OpenClaw Extension 安裝到 PowerToys 的 Command Palette Extensions 資料夾，並重啟 PowerToys。

3. 驗證安裝：

按 Win + 空白鍵 呼叫 Command Palette，輸入 claw 應能看到 OpenClaw 的指令列表。

💡 常用快捷指令：
  • claw ask [問題] — 直接向 AI 提問
  • claw summarize — 摘要剪貼簿內容
  • claw remind [時間] [事項] — 設定提醒
  • claw translate — 翻譯剪貼簿文字

🔧 實戰工作流：三個 Windows 工程師最常用的場景

場景一：快速問 AI 不切換視窗

在寫程式途中遇到問題，直接 Win+空白鍵，輸入 claw ask 這個 error 是什麼意思，AI 回答完直接關掉，繼續寫程式。整個過程不超過 10 秒，不會打斷心流。

場景二：自動整理會議通知

手機配對後，OpenClaw 能讀取你的 Teams/Outlook 通知。每天早上，問 AI：「今天有哪些重要通知和會議？」，它會整理成摘要，不用一個個 App 翻。

場景三：搭配 Claude Code 做 code review

在 VS Code 選取一段程式碼，複製後，按 Win+空白鍵 輸入 claw ask 幫我 review 這段 code，AI 直接讀取剪貼簿內容分析，回覆貼回 VS Code。不需要開新的聊天視窗。

☁️ 進階：把 Gateway 部署到 VPS

如果你希望 AI 助理 24 小時在線（不依賴 PC 是否開著）、或多台裝置共用同一個 Gateway，可以把 OpenClaw 部署到 VPS。

推薦方式是用 DigitalOcean 的 Droplet（最低 $6/月），或直接跑在你家的舊電腦上：

# 在 Ubuntu VPS 上安裝
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
npm install -g openclaw
openclaw init

# 用 PM2 保持背景執行
npm install -g pm2
pm2 start "openclaw start" --name openclaw
pm2 startup
pm2 save

部署完成後，在 Windows 的 OpenClaw 設定中，把 Gateway URL 指向你的 VPS IP，Windows 本機只跑 System Tray + PowerToys 延伸，計算全丟給 VPS。

❓ 常見問題 FAQ

OpenClaw Windows 版本免費嗎？

OpenClaw 本身是開源免費軟體（MIT 授權），openclaw-windows-node 也是開源免費。費用只來自你選用的 AI 模型 API。若用 Ollama 本地模型，完全零成本。

跟 Mac 版 OpenClaw 有什麼差異？

核心 Gateway 功能完全相同。Windows 版額外有 System Tray 常駐、PowerToys Command Palette 整合，讓 Windows 用戶可以用熟悉的快捷鍵呼叫 AI，是專為 Windows 工作流最佳化的設計。

一定要有 VPS 嗎？

不需要。OpenClaw Gateway 可以直接跑在你的 Windows PC 上。VPS 只在需要「電腦關機 AI 還在線」或「多裝置共用」的情況才需要。

PowerToys 整合失敗怎麼辦？

最常見原因是 PowerToys 版本太舊（需要 v0.88+）或 PowerShell 沒有以系統管理員身分執行。更新 PowerToys 後重新跑 openclaw-windows install-powertoys-extension，通常都能解決。

在 Microsoft Build 2026 上展示了哪些功能？

主舞台 Demo 展示了：System Tray 常駐、Node 手機配對（掃 QR 碼即連）、PowerToys Command Palette 呼叫 AI 的完整工作流。Peter Steinberger 親自上台說明 Gateway 架構設計理念。
