OpenCode 三種付費方式快速說明

許多人以為 OpenCode 是「完全免費」的——它的開源本體確實是 MIT 授權免費下載，但實際使用時你仍需要選擇一個計費方式來支付 AI 模型的推理成本：

⚡ Zen
按需付費
只在發送請求時計費，無月費。適合低頻使用者。
🚀 Go
$10/月訂閱
月付訂閱，固定費用，提供更高使用上限。適合中度使用者。
🔑 BYOK
自帶 API Key
直接連接 Anthropic/OpenAI/DeepSeek 等 API。成本最低但需自行管理。
💡 重要說明： OpenCode 的 Zen 和 Go 是 OpenCode.ai 官方提供的代管計費服務，幫你管理 API 連接和費用。自帶 API Key 方式則完全繞過此系統，直接用你自己的 API 帳號。

⚡ Zen 按需付費詳解

Zen 是 OpenCode 官方提供的「零訂閱」入門方式。你只需要在終端機執行 opencode auth，授權後即可使用。沒有月費，系統只在你發送請求時才計費。

Zen 的計費邏輯

Zen 採用「按請求計費」（pay-per-request），每次你發送一個問題給 AI，就對應一次請求。費用根據使用的模型和 token 量決定，由 OpenCode 代理到 Anthropic 服務。

根據 r/opencodeCLI 社群討論，Zen 的大概費率（以 Claude Sonnet 4.5 為基準）：

  • 一次簡短代碼審查（~1000 tokens）：約 $0.003
  • 一次中等複雜度的實作任務（~5000 tokens）：約 $0.015
  • 一次深度架構分析（~20000 tokens）：約 $0.06

什麼時候 Zen 最划算？

  • 你只是偶爾用 OpenCode 測試功能（每月少於 10 次對話）
  • 你想先試用再決定是否升級 Go
  • 你的使用模式不固定（有些月用很少，有些月用很多）

什麼時候 Zen 不划算？

  • 你每週都在密集使用 OpenCode（此時 Go 的月費更可預測）
  • 你需要長上下文對話（token 用量快速累積，超過 $10/月輕鬆達到）

🚀 Go 訂閱方案詳解

Go 方案是 OpenCode 的主要訂閱服務，定價為 $5 第一個月，之後 $10/月。相比 Claude Code 的 $20/月，已經直接省了一半。

Go 的使用限制

根據 OpenCode 官方文件，Go 方案提供「更高的請求上限」，適合每天都在使用 OpenCode 工作的工程師。具體上限數字 OpenCode 未公開，但社群回報超出後的超額費用計算方式與 Zen 相同。

🔑 自帶 API Key（BYOK）詳解

這是最有彈性、也是最能省錢的方式：你自己去申請 AI 供應商的 API Key，然後在 OpenCode 的設定中填入。OpenCode 不會對此收任何額外費用，你只需付 API 供應商的實際用量費用。

設定方式（以 Claude API 為例）

# 在 ~/.config/opencode/config.json 設定
{
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-YOUR_KEY_HERE"
    }
  },
  "model": "claude-sonnet-4-5"
}

常見問題 FAQ

Q1：OpenCode Zen 和 Go 有什麼差別？

Zen 是按需付費（pay-per-request），適合低頻使用者，沒有月費。Go 是訂閱制（$10/月），提供更高使用上限，適合每天使用的工程師。自帶 API Key 則完全繞過 OpenCode 計費，成本最低但需要自行管理 API 帳號。

Q2：OpenCode 可以完全免費使用嗎？

可以。OpenCode 本體是 MIT 授權免費下載。配合 Ollama 本地模型（免費），你可以零費用使用 OpenCode，但本地模型的能力不如 Claude/GPT 等雲端模型。OpenRouter 也有部分免費 tier 可以試用。

Q3：自帶 API Key 比 Go 訂閱方案便宜嗎？

取決於模型和使用量。用 Claude API 的話，輕度到中度使用大約相當。但若改用 DeepSeek V3 API（$0.27/M tokens），成本可以降到 $1–5/月，比 Go 便宜 50–80%。

Q4：OpenCode Zen 支援哪些 AI 模型？

Zen 和 Go 透過 OpenCode 代管服務，預設使用 Claude Sonnet 系列。自帶 API Key 方式可以使用 Claude（Anthropic）、GPT 系列（OpenAI）、Gemini（Google）、DeepSeek、Ollama 本地模型，以及 OpenRouter 上的 100+ 個模型。

Q5：OpenCode 和 Claude Code 哪個比較省錢？

OpenCode 配合自帶 API Key（尤其是 DeepSeek）可以比 Claude Code $20/月 省 70–95%。即使選擇 Go $10/月 方案，也省了 50%。
