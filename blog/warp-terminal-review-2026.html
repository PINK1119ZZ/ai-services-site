什麼是 Warp Terminal？

Warp Terminal 是用 Rust 從零打造的現代終端機，2022 年推出後快速成長，2026 年 5 月已達 54,934 GitHub stars，本週單週新增 28,493 stars 登上 GitHub Trending #1。

🎯 核心特色

  • AI Command Search：自然語言搜尋指令（「如何找出最大的檔案」→ du -sh * | sort -h）
  • Blocks：每個指令輸出獨立區塊，可折疊、分享、搜尋
  • Workflows：儲存常用指令組合，團隊共享
  • 團隊協作：分享終端機 session、指令歷史
  • 極速效能：Rust + GPU 加速，大量輸出不卡頓

官網：warp.dev

核心功能詳解

1. AI Command Search — 自然語言找指令

按 Ctrl + ` 開啟 AI 搜尋，輸入「列出所有 Docker 容器」，Warp 會建議：

docker ps -a

支援繁中輸入，但建議用英文描述更精準。背後使用 GPT-4 模型。

2. Blocks — 指令輸出區塊化

每個指令執行後，輸出會變成獨立「Block」：

  • 點擊 Block 右上角可折疊、複製、分享連結
  • 搜尋功能只搜尋當前 Block，不會被其他輸出干擾
  • 長輸出自動折疊，避免畫面爆炸

3. Workflows — 指令模板

把常用指令存成 Workflow，例如：

# Git 快速提交
git add .
git commit -m "{{commit_message}}"
git push origin {{branch}}

執行時會跳出輸入框填 commit_message 和 branch，團隊可共享 Workflow。

4. 團隊協作

  • Drive：雲端儲存指令歷史、Workflows、設定
  • Session Sharing：分享終端機畫面給隊友（類似 tmux attach）
  • Notebooks：把指令組合成可執行文件，附註解

安裝與設定

macOS 安裝

# Homebrew
brew install --cask warp

# 或下載 .dmg
# https://www.warp.dev/download

Linux 安裝

# Ubuntu/Debian
wget https://releases.warp.dev/stable/latest/warp-terminal_amd64.deb
sudo dpkg -i warp-terminal_amd64.deb

# Arch Linux
yay -S warp-terminal

初次設定

  1. 開啟 Warp，註冊帳號（Google / GitHub 登入）
  2. 選擇主題：Settings → Appearance
  3. 匯入 iTerm2 設定：Settings → Import
  4. 設定 AI 快捷鍵：Settings → Keybindings

推薦設定

# 設為預設終端機
chsh -s /Applications/Warp.app/Contents/MacOS/stable/warp

# 同步 zsh 設定
ln -s ~/.zshrc ~/.warp/zshrc

常見問題 FAQ

Q1: Warp Terminal 免費嗎？

A: Personal 方案完全免費，包含 AI Command Search、Blocks、個人 Workflows。Team 方案 $10/月/人，學生免費。

Q2: 支援 Windows 嗎？

A: 目前僅支援 macOS 和 Linux。Windows 版本在開發中（2026 Q3 預計推出）。

Q3: AI 功能需要網路嗎？

A: 是，AI Command Search 需連線到 Warp 伺服器。離線時仍可使用基本終端機功能。

Q4: 可以匯入 iTerm2 設定嗎？

A: 可以，Settings → Import → iTerm2 會自動轉換主題、快捷鍵、Profile。

Q5: tmux 可以在 Warp 裡用嗎？

A: 可以，但部分 Warp 功能（Blocks、AI）在 tmux session 內會失效。建議用 Warp 原生 Session Sharing 取代 tmux。

Q6: 隱私安全嗎？指令會上傳嗎？

A: AI 搜尋時會傳送查詢內容到 Warp 伺服器，但不會上傳完整指令歷史。可在 Settings → Privacy 關閉遙測。
