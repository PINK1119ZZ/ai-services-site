Bot 雲託管跟自己請工程師做差在哪？
自己做一支基礎 Bot 市場行情 NT$8,000-30,000 一次性 + 每月 NT$2,000-5,000 維護。我們月費 NT$1,500 起就包含開發、託管、維護、AI 額度。等於以自己做的 1/3 價格、在 1 天內就能用。
支援哪些平台？
LINE 官方帳號、Telegram（Bot API 和 User API 都支援）、Discord、Slack、Messenger。企業訂製方案可以同時接多平台，客戶一段訊息跨平台同步發送。
TG 群組監控是什麼？可以監哪些東西？
監聽指定 Telegram 群組 / 頻道 / 私訊，偵測關鍵字（如「到期」、「暫停」、「打款」、「下架」）、刪除消息、超時未回覆，即時推送預警到指定群或窗口。也可以把每條訊息自動記錄到 Google Sheet 做歸檔。適合客服管理、廣告代理、外事號監控、電商商戶管理等需要多窗口監控的場景。
爬蟲 Bot 可以爬什麼？違法嗎？
公開網頁、電商商品頁、論壇、競品介紹頁、公告頁等「對外公開」的資料都可以。我們不爬需要登入的私密頁、不做刷點擊刷流量、不繞過付費牆。所有爬蟲都遵守 robots.txt 和合理速率。
不會寫程式也能用嗎？
完全可以。所有 Bot 都有客戶後台，你只要在網頁填「關鍵字 → 回覆」、上傳公司資料、選觸發條件，我們幫你搞定所有技術部署。AI 客服也只是填一段「公司介紹 + 回覆風格」。
可以隨時取消嗎？資料會不會遺失？
完全沒有綁約，隨時可取消，下個月不再扣款。取消後所有對話紀錄、設定、Sheet 資料可以完整打包匯出給你。
我的客戶資料會不會被外洩或用來訓練 AI？
每位客戶獨立的資料庫分區，對話紀錄只有你自己能看到。我們簽 NDA，絕不讀取、轉售、或用於訓練任何 AI 模型。企業方案另外提供獨立伺服器選項。
