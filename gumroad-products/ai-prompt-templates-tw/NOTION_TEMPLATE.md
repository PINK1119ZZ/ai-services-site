# Notion 模板導入說明

## 如何在 Notion 中使用這套 Prompt 模板庫？

### 方法 A：直接貼入（最快）

1. 打開你的 Notion 頁面
2. 輸入 `/` → 選擇「Code」區塊
3. 將 Prompt 原文貼入 Code 區塊（保留格式，方便複製）
4. 在 Code 區塊上方加一個「Text」區塊說明情境

### 方法 B：建立 Prompt 資料庫（建議）

建立一個 Notion Database，欄位如下：

| 欄位名 | 類型 | 說明 |
|--------|------|------|
| 名稱 | Title | Prompt 名稱 |
| 類別 | Select | 商業 / 行銷 / 開發者 / 創意 |
| 適用 AI | Multi-select | ChatGPT / Claude / Gemini |
| 效果評分 | Number | 自己試用後打 1-5 分 |
| 最後使用 | Date | 追蹤使用頻率 |
| Prompt 原文 | Text | 完整 Prompt |
| 輸出範例 | Text | 貼上你實際的輸出 |
| 備注 | Text | 使用心得或修改版本 |

### 方法 C：使用頁面分類（簡單）

在 Notion 建立 4 個子頁面：
- 📊 商業類（01-08）
- 📣 行銷類（09-16）  
- 💻 開發者類（17-23）
- ✍️ 創意類（24-30）

每頁建立一個 Toggle List，每個 Toggle = 一個 Prompt，裡面放原文和範例。

---

## 小技巧：讓 Prompt 效果更好

1. **填空前先思考** — 每個 [填寫] 都值得思考 30 秒，想越清楚，輸出越好
2. **保存好的輸出** — 看到好的輸出，立即存到 Notion，下次可以當「範例」給 AI 參考
3. **建立個人變體** — 試用幾次後，在 Prompt 下方建立「我的客製化版本」
4. **比較不同 AI** — 同一個 Prompt 在 Claude / ChatGPT / Gemini 的輸出差異很有趣
