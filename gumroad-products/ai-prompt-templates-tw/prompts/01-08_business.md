# 商業類 Prompt 模板（8 個）
> 適合：創業者、商業分析師、業務、PM、企業主

---

## Prompt 01：商業計劃書框架

**情境**：你需要快速產出一份商業計畫書，但不知道從何開始。

**Prompt**：
```
你是一位有 15 年創投審查經驗的 VC 分析師。請幫我為以下創業構想撰寫一份精簡商業計畫書框架：

構想：[在此填寫你的創業構想，1-2 句話]
目標市場：[在此填寫，例：台灣中小企業主，20-45 歲]
解決問題：[在此填寫核心痛點]

輸出格式（Markdown）：
1. 執行摘要（150 字以內）
2. 問題與解決方案（各 100 字）
3. 目標市場規模（TAM/SAM/SOM）
4. 商業模式（收費方式 + 單位經濟學）
5. 競爭優勢（3 點）
6. 12 個月里程碑（6 個節點）
7. 資金需求概要

請使用繁體中文，語氣專業但易懂。
```

**輸出範例**（以 AI 工具 SaaS 為例）：
```
## 執行摘要
AutoAssist 是面向台灣 SMB 的 AI 客服自動化平台，幫助每月客服請求量 500+ 的中型電商將人工客服成本降低 60%。首年目標鎖定台北大台北區 200 家電商業者，MRR 目標 NT$1.5M...

## 問題
台灣中型電商每月客服量龐大，但人力成本高漲；使用 AI 的門檻又太高（需要工程師）...
```

**進階變體**：
- 加入「請以台灣法規（公司法、消費者保護法）為前提調整合規說明」
- 加入「請加入台灣投資人最在意的 3 個風險與應對策略」

## Prompt 02：競品分析報告

**情境**：需要快速了解 2-3 個競品的優劣勢。

**Prompt**：
```
你是一位資深策略顧問。請對以下競品進行分析：

我的產品：[填寫你的產品]
競品 A：[填寫競品名稱與簡介]
競品 B：[填寫競品名稱與簡介]

請從以下 5 個維度分析每個競品（每個維度 100 字內）：
1. 定價策略與商業模式
2. 目標客群與訴求
3. 核心功能優勢
4. 已知弱點或用戶抱怨
5. 對我方產品的威脅程度（1-10 分）

最後給出：「我方的差異化策略建議」（200 字以內）

語言：繁體中文
格式：Markdown 表格 + 分析段落
```

**進階變體**：
- 「請加入 Google Trends 搜尋量趨勢分析（我無法直接查詢，請假設合理數據並標注）」
- 「請以台灣市場為主，加入台灣特有的競品（如：91APP、SHOPLINE）」

## Prompt 03：市場調查報告摘要

**情境**：你有一份原始調查數據，需要快速整理成可呈現的報告。

**Prompt**：
```
你是一位市場研究分析師。我有以下調查數據，請幫我整理成結構化報告摘要：

調查數據：
[在此貼上你的原始數據，可以是表格、列表、或段落]

輸出格式：
1. 三句話執行摘要
2. 五大關鍵發現（每條 1-2 句，附對應數據支撐）
3. 機會點（3 個，每個 50 字）
4. 風險警示（2 個）
5. 建議下一步行動（3 條）

請使用正式繁體中文，適合向管理層報告的語氣。
```

## Prompt 04：銷售開發信（Cold Email）

**情境**：需要寫一封有高開信率的 B2B 開發信。

**Prompt**：
```
你是一位 B2B SaaS 業務頂尖高手，擅長寫讓 CEO / 採購主管開信的冷開發信。

我的產品/服務：[填寫]
目標收件人職稱/公司類型：[填寫，例：台灣中型製造業的 IT 主管]
核心價值主張（一句話）：[填寫]
社會證明（已有的客戶或數據）：[填寫，若無可留空]

請寫 3 個版本的開發信，每個版本不超過 150 字：
- 版本 1：痛點切入（以問題開頭）
- 版本 2：數據切入（以驚人數字開頭）
- 版本 3：社會證明切入（以客戶案例開頭）

每個版本包含：主旨行 + 信件正文 + CTA（行動呼籲）

語言：正式繁體中文
```

**輸出範例**（版本 1 節選）：
```
主旨：[公司名稱]的客服費用，每月少付 60%？

[收件人姓名] 您好，

台灣電商平均每月客服成本佔營收 8-12%。AutoAssist 已幫助 30 家電商將這個比例降至 3% 以內——平均 90 天回本。

可以給我 15 分鐘，分享我們怎麼做到的嗎？

AutoAssist 業務團隊 敬上
```

## Prompt 05：提案書／簡報大綱

**情境**：需要在 30 分鐘內產出一份客戶提案書大綱。

**Prompt**：
```
你是一位擅長策略性敘事的提案顧問。請為以下情境生成一份完整的提案書大綱：

提案對象：[填寫，例：台灣中型零售連鎖品牌的行銷長]
提案目的：[填寫，例：說服對方採購我們的 AI 行銷自動化方案]
預算範圍：[填寫，例：NT$50 萬年費]
競爭狀況：[填寫，例：對方正在評估 3 家供應商]

大綱格式（10 頁投影片結構）：
- 封面頁：標題建議
- P2-3：現況痛點（用數字量化）
- P4-5：解決方案說明
- P6：ROI 計算（投資回報）
- P7：執行時程
- P8：成功案例
- P9：定價方案（3 個選項）
- P10：下一步行動

每頁提供：標題 + 3-4 個重點 bullet + 推薦視覺元素

語言：繁體中文
```

## Prompt 06：OKR 設定輔助

**情境**：季度初需要幫團隊設定 OKR（目標與關鍵結果）。

**Prompt**：
```
你是一位精通 OKR 方法論的組織顧問（Google/Spotify 等企業背景）。

請幫我為以下情境設定一套 Q3 2026 OKR：

公司階段：[填寫，例：種子輪後，15 人團隊]
季度核心目標（方向）：[填寫，例：驗證台灣市場的付費意願]
現有數據基準：[填寫，例：目前 MAU 500，付費用戶 30，MRR NT$15 萬]

請輸出：
- 1 個公司級 Objective（鼓舞人心的方向）
- 3 個 Key Results（可量化，有挑戰但可達成）
- 每個 KR 附：基準值、目標值、衡量方式、負責單位

附：3 個常見 OKR 錯誤提醒（讓我避開）

語言：繁體中文
```

## Prompt 07：合約條款草稿（基礎）

**情境**：需要一份基礎的服務合約或保密協議模板。

**⚠️ 注意**：此模板僅供參考，正式合約請諮詢律師。

**Prompt**：
```
你是一位熟悉台灣商業法規（公司法、民法、著作權法）的法律顧問。

請幫我起草一份基礎的服務合約，涵蓋以下條款：

合約類型：[填寫，例：SaaS 軟體服務訂閱合約]
甲方：[服務提供商名稱]
乙方：[客戶類型，例：企業法人]
服務內容：[填寫]
服務期限：[填寫]
費用：[填寫]

必要條款：
1. 服務範圍定義
2. 費用與付款方式
3. 智慧財產權歸屬
4. 保密義務
5. 違約與終止條款
6. 免責聲明

請以繁體中文輸出，格式使用正式法律文件樣式。
每個條款後請加注「【風險提醒】」說明可能的法律爭議點。
```

## Prompt 08：產品說明文案（電商/官網）

**情境**：需要撰寫轉換率高的產品說明頁文案。

**Prompt**：
```
你是一位擅長台灣消費者心理的電商文案師，熟悉 AIDA（注意→興趣→欲望→行動）框架。

產品名稱：[填寫]
產品類型：[填寫，例：AI 寫作工具年訂閱]
目標客群：[填寫，例：台灣部落客、Youtuber，25-40 歲]
核心功能（3 個）：[填寫]
定價：[填寫]
競品比較優勢：[填寫]

請輸出完整產品說明文案，包含：
1. 主標題（吸引注意，10 字以內）
2. 副標題（說明價值主張，20 字以內）
3. 痛點描述段落（100 字，引起共鳴）
4. 功能介紹（3 個 Feature + Benefit 格式，每個 50 字）
5. 社會證明位置（建議放什麼類型的見證）
6. 定價區塊文案
7. CTA 按鈕文字（3 個選項）

語言：台灣繁體中文，口語親切
```
