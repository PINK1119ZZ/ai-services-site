# 行銷類 Prompt 模板（8 個）
> 適合：行銷人、文案寫作者、社群媒體經理、電商店主、品牌主

---

## Prompt 09：SEO 文章框架

**情境**：需要快速產出一篇 SEO 友善的文章結構。

**Prompt**：
```
你是一位有 8 年 SEO 寫作經驗的內容策略師，熟悉台灣繁體中文搜尋市場。

目標關鍵字：[填寫主要關鍵字，例：台灣 AI 工具推薦 2026]
文章類型：[填寫，例：評測比較文 / 教學文 / 清單文]
目標讀者：[填寫，例：台灣 30-45 歲想提升工作效率的上班族]
競品文章弱點：[填寫，例：競品文章沒有台灣特化內容，沒有定價資訊]

請輸出：
1. SEO 優化標題（含主關鍵字，60 字以內）
2. Meta Description（含次關鍵字，155 字以內）
3. 文章大綱（H1-H3 結構，含建議字數）
4. 內部連結建議（3 個相關文章主題）
5. 外部引用建議（3 個可信來源類型）
6. 目標長尾關鍵字清單（10 個，自然分散在大綱中）
7. FAQ 問題清單（6 個，適合 Schema Markup）

請特別注意台灣繁中使用者的搜尋習慣與語言習慣。
```

## Prompt 10：社群媒體貼文（Facebook / Threads）

**情境**：需要為一篇文章或產品發布撰寫多平台貼文。

**Prompt**：
```
你是一位台灣社群媒體專家，了解台灣 Facebook 用戶（35-55 歲）和 Threads 用戶（25-40 歲）的閱讀習慣差異。

貼文主題：[填寫，例：AI 工具幫我把每週工作時間縮短 10 小時]
核心訊息：[填寫，例：使用 3 個 AI 工具後的真實心得]
附件類型：[填寫，例：1 張截圖 / 無圖片 / 影片]

請分別輸出：

【Facebook 版本】（300-500 字，可加 emoji，適合中年族群）
- 開頭：引發共鳴的問題或故事
- 中段：具體案例與數據
- 結尾：互動呼籲（問問題讓人留言）
- 標籤建議（3-5 個）

【Threads 版本】（100-200 字，簡潔有力，適合年輕族群）
- 開頭：一句吸睛金句
- 中段：2-3 個 bullet
- 結尾：CTA 或問題

語言：台灣口語繁體中文，自然親切
```

## Prompt 11：Google Ads 廣告文案

**情境**：需要撰寫 Google 搜尋廣告文案（符合字數限制）。

**Prompt**：
```
你是一位 Google Ads 認證廣告師，精通台灣市場的廣告優化。

產品/服務：[填寫]
目標關鍵字（廣告觸發字）：[填寫]
主要競品（了解差異化）：[填寫]
核心 USP（獨特賣點）：[填寫]
Landing Page 核心內容：[填寫]

請輸出 3 組 RAS（回應式搜尋廣告）文案：

每組包含：
- 標題 1（30 字以內）×3 個版本
- 標題 2（30 字以內）×3 個版本  
- 標題 3（30 字以內）×3 個版本
- 說明文字（90 字以內）×2 個版本

每組策略角度不同：
- 版本 A：痛點解決型
- 版本 B：社會證明型
- 版本 C：緊迫感 / 限時優惠型

使用台灣繁中，所有字數需符合 Google 規範。
```

## Prompt 12：品牌故事文案

**情境**：需要撰寫官網「關於我們」或品牌故事頁面。

**Prompt**：
```
你是一位品牌敘事專家，擅長寫能讓台灣消費者產生情感共鳴的品牌故事。

品牌/公司名稱：[填寫]
創辦年份：[填寫]
創辦故事（1-2 句）：[填寫，可以是遇到的問題或動機]
核心價值觀（3 個）：[填寫]
里程碑（2-3 個值得提的成就）：[填寫]
目標客群：[填寫]

請輸出「關於我們」頁面文案：
1. 英雄式開場（100 字，以創辦人的掙扎或洞察開頭）
2. 為何而生（150 字，解釋品牌存在的理由）
3. 我們的承諾（100 字，3 條具體承諾）
4. 里程碑時間軸文案（每個里程碑 30 字）
5. 邀請加入（50 字 CTA）

語氣：真誠、有溫度、不誇大，適合台灣消費者閱讀
```

## Prompt 13：Email 行銷 序列（Welcome Sequence）

**情境**：新訂閱者加入後，需要一套 5 封的歡迎郵件序列。

**Prompt**：
```
你是一位 Email 行銷專家，精通 ConvertKit / beehiiv 等平台的自動化序列設計。

品牌：[填寫]
訂閱者類型：[填寫，例：對 AI 工具有興趣的台灣部落客]
最終目標：[填寫，例：引導購買 $29 的 Prompt 模板包]
品牌語氣：[填寫，例：專業但親切，像業界朋友分享經驗]

請設計 5 封歡迎序列郵件：

【Email 1 - 立即發送】歡迎 + 兌現承諾
【Email 2 - Day 2】提供一個「快速勝利」的技巧
【Email 3 - Day 4】分享一個真實故事 / 案例研究
【Email 4 - Day 7】克服反對意見 + 軟性推薦
【Email 5 - Day 10】直接銷售郵件

每封郵件包含：
- 主旨行（含 A/B 測試版本）
- Preview Text（50 字）
- 正文大綱（重點 bullets，非完整文章）
- CTA 按鈕文字

語言：台灣口語繁體中文，不要太正式
```

## Prompt 14：Podcast 腳本框架

**情境**：需要為一集 30 分鐘的 Podcast 準備腳本。

**Prompt**：
```
你是一位資深 Podcast 製作人，了解台灣聽眾的收聽習慣。

節目名稱：[填寫]
本集主題：[填寫]
主持人人數：[填寫，例：1 人獨播 / 2 人對談]
目標聽眾：[填寫]
核心論點（你想傳達的 1-2 個關鍵訊息）：[填寫]

請輸出 30 分鐘 Podcast 腳本框架：

00:00-02:00 開場鉤子（一個讓人繼續聽的問題或驚人陳述）
02:00-05:00 本集預告（今天要討論什麼，聽完有什麼收穫）
05:00-20:00 主體內容（3 個分段，每段 5 分鐘，含轉場語）
20:00-25:00 案例分享或訪談（可留空讓主持人填充）
25:00-28:00 行動清單（3-5 個聽完可做的事）
28:00-30:00 結尾 + 訂閱呼籲

每個分段提供：開場白建議 + 3-4 個討論點 + 轉場語句

語言：自然的台灣口語，適合唸出來
```

## Prompt 15：Threads / X（Twitter）貼文系列

**情境**：需要一系列可串成「Thread」的短貼文。

**Prompt**：
```
你是一位 Threads / X 內容策略師，了解如何用 Thread 格式教學並建立粉絲。

主題：[填寫，例：10 個 AI 工具讓我的工作效率提升 3 倍]
核心論點：[填寫]
受眾：[填寫，例：台灣 30-45 歲想學 AI 的上班族]

請設計一個 10 篇的 Thread 系列：

Tweet 1：主貼文（鉤子，讓人想看後續）
Tweet 2-8：逐一展開論點（每篇 100-150 字，帶 emoji，含 1 個行動建議）
Tweet 9：彙整摘要（可截圖的重點清單）
Tweet 10：CTA + 互動呼籲

規則：
- 每篇台灣繁中，100-150 字
- 必要時加台灣在地化範例（APP、工具、情境）
- 最後有分享呼籲和 Follow CTA
```

## Prompt 16：產品發布公告文案

**情境**：新產品上線，需要跨平台的公告文案。

**Prompt**：
```
你是一位 Product Marketing 專家，熟悉台灣 SaaS / 數位產品的發布溝通。

產品名稱：[填寫]
發布日期：[填寫]
核心功能（3 個）：[填寫]
解決什麼問題：[填寫]
定價：[填寫]
Early Bird 優惠（若有）：[填寫]

請輸出以下發布文案：

1. Press Release 標題 + 副標題（媒體稿格式）
2. 官方網站公告（200 字）
3. Email 公告信（寄給現有用戶，300 字）
4. Facebook 發文（300 字，適合分享）
5. Threads 發文（150 字，簡潔）
6. LinkedIn 發文（300 字，專業語氣）

每個版本的語氣和焦點可以不同，但核心訊息一致。
語言：台灣繁體中文
```
