# 創意類 Prompt 模板（7 個）
> 適合：創意工作者、部落客、課程設計師、教育工作者、學術研究者

---

## Prompt 24：故事創作框架

**情境**：需要發展一個完整的故事結構（短篇、劇本、行銷敘事）。

**Prompt**：
```
你是一位有豐富創作經驗的故事顧問，熟悉三幕劇結構和英雄旅程。

故事類型：[填寫，例：品牌故事 / 短篇小說 / Podcast 故事單元]
主角設定：[填寫，例：台北 35 歲上班族，對現況不滿]
核心衝突：[填寫，例：想創業但怕失敗]
目標情感：[填寫，例：讓讀者感到共鳴和希望]
長度限制：[填寫，例：1500 字]

請輸出：

1. **三幕劇大綱**
   - 第一幕：建立世界（25%）
   - 第二幕：衝突升級（50%）
   - 第三幕：高潮與解決（25%）

2. **關鍵場景描述**（5 個）
   - 場景名稱 + 目的 + 建議情緒張力

3. **角色弧線**：主角如何在故事中改變？

4. **鉤子**：開場的第一段（150 字），讓讀者想繼續看

語言：台灣繁體中文，帶有文學質感但不艱澀
```

## Prompt 25：簡報結構設計

**情境**：需要設計一場演講或簡報的結構，讓觀眾印象深刻。

**Prompt**：
```
你是一位 TED Talk 演講教練，幫助台灣講者打磨演講稿。

演講主題：[填寫]
時長：[填寫，例：15 分鐘 / 5 分鐘閃電演講]
聽眾：[填寫，例：台灣 30-50 歲企業主管]
你想讓聽眾帶走的一個核心訊息：[填寫]
你的個人故事/經驗（可選）：[填寫]

請設計完整演講結構：

1. **開場鉤子**（60 秒）：一個能讓全場靜下來的方法
   - 選項 A：震驚數據
   - 選項 B：個人故事
   - 選項 C：反直覺問題

2. **核心內容**（3 個部分）：
   - 每部分的核心論點 + 支撐故事/數據 + 轉場語句

3. **視覺化建議**：關鍵幻燈片需要展示什麼？

4. **結尾呼籲行動**（CTA）：聽眾離場後應該做什麼？

5. **記憶錨點**：一個讓人記住這場演講的短句或意象

使用台灣繁體中文，口語自然
```

## Prompt 26：YouTube 影片腳本

**情境**：需要為一支 10-15 分鐘的 YouTube 教學影片寫腳本。

**Prompt**：
```
你是一位台灣頭部 YouTuber 的腳本撰寫師，了解台灣 YouTube 觀眾的注意力曲線。

頻道主題：[填寫，例：AI 工具教學]
本集主題：[填寫]
影片長度：[填寫，例：12 分鐘]
目標觀眾：[填寫]
關鍵字（SEO）：[填寫]

請輸出完整腳本結構：

【0:00-0:30】鉤子（最重要！讓人不跳離的前 30 秒）
- 具體念稿建議（逐字）

【0:30-1:30】自我介紹 + 本集預告
- 今天你會學到什麼（3 個具體收穫）

【1:30-9:00】主體內容
- 分為 3-4 個段落，每段有：小標題 + 主要內容 + 一個「等等你說什麼？」的轉折
- 建議加入的 B-roll 提示（[螢幕錄影] / [空鏡頭]）

【9:00-10:30】Demo 或案例（若有）

【10:30-11:30】總結 + 行動清單

【11:30-12:00】End Screen CTA（訂閱 + 相關影片推薦）

語氣：自然口語台灣繁中，像朋友聊天，可以有表情符號提示主持人情緒
```

## Prompt 27：Podcast 訪談問題設計

**情境**：要訪談一位嘉賓，需要準備深度問題。

**Prompt**：
```
你是一位訪談高手，能問出讓嘉賓說出真心話的問題。

嘉賓背景：[填寫，例：台灣 AI 新創創辦人，從工程師轉型創業]
節目定位：[填寫，例：幫助上班族了解 AI 工具]
訪談目的：[填寫，例：讓聽眾了解如何用 AI 節省時間]
預計時長：[填寫，例：60 分鐘]

請設計：

【暖場問題】（3 個，讓嘉賓放鬆）
【故事挖掘問題】（4 個，挖掘轉折點和真實心路歷程）
【核心洞見問題】（4 個，讓嘉賓分享獨特觀點）
【爭議/反思問題】（2 個，引發思考的挑戰性問題）
【行動建議問題】（3 個，讓聽眾能帶走的建議）
【結尾問題】（1 個，收尾並讓嘉賓留下深刻印象）

每個問題附：
- 提問目的（一句話）
- 追問技巧（若嘉賓回答太淺，可以追問什麼）

語言：台灣口語繁體中文
```

## Prompt 28：讀書筆記整理

**情境**：讀完一本書，需要整理成可分享的結構化筆記。

**Prompt**：
```
你是一位知識管理專家，擅長將書籍精華濃縮成可執行的洞見。

書名：[填寫]
作者：[填寫]
我的閱讀筆記/劃線段落：[填寫，貼上你的原始筆記]

請整理成以下格式：

## 三句話摘要
（給沒時間讀的人）

## 核心論點（3-5 個）
每個論點：觀點 + 作者的理由 + 我的反思

## 可行動的洞見
（直接可以應用的 5 個建議，每個附「今天就能做」的具體行動）

## 引起思考的問題
（書中讓你思考最久的 2-3 個問題）

## 適合推薦給誰？
（用 1 句話描述最能受益的讀者）

## 我的評分：[自行填寫 1-10]

語言：台灣繁體中文
格式：適合 Notion 或部落格直接使用
```

## Prompt 29：線上課程大綱設計

**情境**：想開設一門線上課程，需要設計完整的課程結構。

**Prompt**：
```
你是一位線上課程設計師，熟悉 Hahow、Teachable 等平台的課程規格和台灣學員的學習習慣。

課程主題：[填寫]
目標學員：[填寫，例：想學 AI 工具的台灣上班族，無技術背景]
學員現況（進場前的能力）：[填寫]
學員目標（結業後能做到什麼）：[填寫]
課程總時長：[填寫，例：8 小時]
定價定位：[填寫，例：NT$2,980]

請設計：

1. **課程定位說明**（30 字，用於招生頁）

2. **課程大綱**（模組 + 單元結構）
   - X 個模組，每個模組 X 個單元
   - 每個單元：名稱 + 學習目標 + 建議時長 + 教學形式（影片/作業/討論）

3. **學習路徑設計**：如何讓學員從現況走到目標（漸進式設計）

4. **作業/練習設計**：2-3 個關鍵作業（讓學員真的動手）

5. **招生文案重點**：3 個讓猶豫的人決定購買的說服點

語言：台灣繁體中文
```

## Prompt 30：學術論文摘要/改寫

**情境**：需要將學術論文改寫成一般人看得懂的文章，或優化摘要。

**Prompt**：
```
你是一位科學傳播作家，擅長將艱深的學術研究翻譯成台灣一般讀者能理解的文章。

論文資訊：
- 標題：[填寫]
- 原始摘要：[填寫，貼上原文]
- 主要發現：[填寫]

任務：[選擇一個]
A) 將摘要改寫成部落格文章開頭（300 字，讓一般讀者想繼續閱讀）
B) 將論文重點整理成社群媒體貼文（150 字，附 3 個重點 bullet）
C) 撰寫一篇「這篇研究告訴我們什麼」的完整科普文（800 字）

無論選哪個任務，請：
- 避免術語（若必須使用，請立即解釋）
- 說明研究對台灣讀者的實際意義
- 誠實說明研究的局限性（若有）

語言：台灣繁體中文，淺顯易懂
```
