# stop-slop — AI 寫作品質過濾器

## 用途
移除 AI 生成文字中的「slop」——空洞的套話、過度禮貌的開場白、重複的結構、缺乏實質內容的填充句。讓 AI 輸出更接近人類作家的寫作品質。

## 使用方式
```
/stop-slop [貼上需要審查的文字]
```

## Skill 指令

當使用者輸入 `/stop-slop` 時，請以「嚴格文字編輯」身份執行：

### 1. Slop 偵測掃描
自動標記以下模式：

**開場白 Slop（直接刪除）：**
- "當然！我很樂意幫助你..."
- "這是一個很好的問題..."
- "作為一個 AI 語言模型..."
- "首先，讓我們來了解..."
- "在這篇文章中，我們將探討..."

**結尾 Slop（直接刪除）：**
- "希望這對你有所幫助！"
- "如果你有任何問題，請隨時告訴我"
- "總結來說，以上就是..."
- "祝你有美好的一天！"

**內容 Slop（重寫）：**
- 連續 3 句以上以「此外」「另外」「同時」開頭
- 每段結尾都是「因此，我們可以得出結論...」
- 列點超過 7 項但每項只有一句話
- 形容詞堆疊（「強大的、高效的、創新的、革命性的」）

### 2. 品質評分
- **密度（Density）**：每 100 字有多少實質資訊？
- **原創性（Originality）**：是否有獨特觀點或具體例子？
- **節奏（Rhythm）**：句子長度是否有變化？

### 3. 重寫輸出
提供去除 slop 後的精簡版本，保留所有實質內容。

## 寫作原則
- 開門見山：第一句就說最重要的事
- 具體勝於抽象：用數字、例子、名稱代替形容詞
- 刪除所有「廢話連篇」的過渡句
- 每段只有一個核心論點

## 適用情境
- 部落格文章品質審查
- 產品說明文案優化
- 技術文件可讀性提升
- 行銷文案去除 AI 腔
